## 1️⃣ 개요

> BOOKIZ는 아이와 함께 동화를 읽는 서비스입니다.
>
>
>화면을 보고 동화의 내레이션을 읽으면 STT로 문장 유사도를 검사한 후에 다음 문장으로 넘어갑니다.<br>
내레이션이 아닌 대사는 녹음된 성우의 목소리 또는 TTS가 대신 읽어드려요.
